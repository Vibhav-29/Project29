# 🛒 GrociQuest

GrociQuest is a modern and user-friendly **online grocery store web application** designed to simplify your shopping experience. With features like product search, category-wise browsing, and a working cart system, GrociQuest offers a sleek interface built for the web.

---

## 🚀 Features

- 🔍 Real-time search for products
- 🍫 Category pages like Chocolates, Snacks, etc.
- 🛒 Fully functional and modern shopping cart
- 💡 Dark mode option (optional in future updates)
- 📱 Responsive UI optimized for laptops

---

## 💻 Tech Stack

- **Frontend:** HTML, CSS, JavaScript (or Flutter Web if upgraded)
- **Backend:** Node.js (Express)
- **Database:** MongoDB
- **Version Control:** Git & GitHub
